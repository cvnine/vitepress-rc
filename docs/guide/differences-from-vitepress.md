
# 测试

| 参数      | 说明             | 类型    | 可选值 | 默认值 |
| --------- | ---------------- | ------- | ------ | ------ |
| width     | 宽度(最外层)     | number  | -      | -      |
| height    | 高度(最外层)     | number  | -      | -      |
| className | 类(最外层)       | string  | -      | -      |
| style     | 外层样式(最外层) | css     | -      | -      |
| disabled  | 是否可用         | boolean | true   | false  |
